## Run Locally

Clone the project

```bash
  git clone https://github.com/bhrama-br/football-supporter-user.git
```

Go to the project directory

```bash
  cd football_supporter_user
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
